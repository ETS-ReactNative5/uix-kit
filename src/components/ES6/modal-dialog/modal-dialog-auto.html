Modal Dialog Automatically Open

Trigger with a simple JavaScript

$( document ).ready( function() {
	$(document).UixFireModalDialog({
		id: 'modal-open-1',
		height: false,
		width: false,
		speed: 500,
		lightbox  : true,
		autoClose : false //If the value is 2000, it will automatically close after 2 seconds.
	});

} );
